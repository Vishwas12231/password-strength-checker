# 🔐 Password Strength & Crack Time Estimator

An advanced password security analysis tool that estimates:

- Password entropy
- Structural weaknesses
- Estimated crack time under multiple attacker models
- Offline vs Online attack simulation

## 🚀 Features

- Entropy calculation
- GPU attack simulation
- Botnet attack modeling
- Common password detection
- Pattern detection (repeats, sequences)
